The Story So Far

The Beginning
Day Zero

Started with a simple question: "How does this website actually work?" Opened the browser inspector for the first time. Saw a wall of HTML. Got curious. Started Googling. Couldn't stop.

Foundations — Phase 1
Discovering The Odin Project

Found The Odin Project — a free, open-source full-stack curriculum that doesn't hand-hold or cut corners. Started the foundations path. Learned how the web works, how browsers render pages, and what HTTP actually does.

The Odin Project How The Web Works HTTP
Foundations — Phase 2
HTML & CSS Clicked

Built my first real pages. Learned Flexbox and Grid — the moment layouts stopped being confusing. Understood the box model, specificity, responsive design. Started actually caring about how things look.

HTML5 CSS3 Flexbox CSS Grid Responsive Design
Git & GitHub
Version Control Unlocked

Learned Git — one of the most important tools in any developer's toolbox. First commit, first push, first pull request. Understood why developers never work without version control again. Also set up GitLab.

Git GitHub GitLab CLI
First Real Project
Built the Telegram Bot

Stepped outside the curriculum and built something real on my own. Combined gallery-dl and the Telegram Bot API in Python to create a bot that downloads media on demand. My first proof that I can learn something new and ship it independently.

Python Telegram Bot API gallery-dl First Real Project ✓
Right Now — Active
JavaScript Gets Serious

Deep in JavaScript fundamentals — closures, callbacks, async/await, fetch, OOP. Building DOM projects. Writing cleaner code. Getting comfortable with the language that powers the interactive web.

JavaScript DOM API Async / Fetch OOP

Node.js, Express, SQL, React — the full-stack is on the horizon. The plan is to build real web applications that talk to databases, handle user authentication, and deploy to production. Full-Stack incoming.
