My Setup

WHAT
I USE.

Everything I use to write code, learn, and build things. Mostly free tools because I'm a student — but free doesn't mean bad. Some of the best developer tools in the world are completely free.

⌨️ Editor & Terminal
🟦
Visual Studio Code
My main code editor. Fast, extensible, and has excellent support for every language I use. Extensions: Prettier, ESLint, GitLens, Live Server.
Free
🖥️
Terminal / Bash
Command line for Git, running scripts, navigating files, and running Python bots. Once you learn the CLI, going back feels wrong.
Free
🛠️ Development
🐙
Git + GitHub + GitLab
Version control for everything. GitHub for open source, GitLab for hosting this portfolio via GitLab Pages (free static hosting).
Free
🔥
Firefox Developer Edition
My browser of choice for development. The CSS grid inspector and layout tools are genuinely better than Chrome's.
Free
🐍
Python 3
For automation scripts, bots, and anything backend. Used to build my Telegram bot project.
Free
📚 Learning
⚔️
The Odin Project
My primary curriculum for learning full-stack web development. Free, open-source, project-based. The best free coding education on the internet — no shortcuts.
Free
📖
MDN Web Docs
Mozilla's documentation for HTML, CSS and JavaScript. The authoritative reference — I check it almost every day.
Free
💬
Stack Overflow + Discord
Stack Overflow for searching past answers. The Odin Project Discord for community support and motivation.
Free
🎨 Design
🖼️
Figma (learning)
Just starting to learn Figma for UI design and wireframing. Free tier is more than enough for personal projects.
Learning
🔤
Google Fonts
Free web fonts. This portfolio uses Bebas Neue (display), Outfit (body), and JetBrains Mono (code). All free.
Free
☁️ Hosting & Deployment
🦊
GitLab Pages
Free static site hosting built into GitLab. This entire portfolio is hosted here for free. Zero configuration for HTML/CSS/JS projects.
Free

I deliberately use free tools wherever possible — not because I can't afford paid ones, but because the best developer tools in the world are free and open source. Learning on free tools teaches you more. When you can't rely on magic buttons, you understand what's actually happening under the hood.
