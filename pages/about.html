Who I Am

ABOUT
ME.

Hey — I'm Siamuddin, a self-taught developer from Bangladesh 🇧🇩. I'm learning full-stack web development through The Odin Project — one of the most rigorous free coding curricula on the internet.

My journey started with curiosity and a browser — I wanted to understand how websites actually work under the hood. So I started learning. I haven't stopped since.

Right now I'm focused on JavaScript fundamentals, HTML/CSS mastery, and building real things that solve real problems. My first project was a Telegram media bot — a Python automation tool that combines two APIs to let users download media directly into Telegram. Small, but mine.

My goal is to become a full-stack developer who can take an idea from a napkin sketch to a live, deployed product. I love clean code, interesting problems, and websites that feel genuinely good to use.

How I Work

🔨
Build First

I learn by doing. Reading docs is fine, but the real understanding comes when I open a blank file and start from nothing. Every bug is a lesson.

🔍
Understand Deep

I don't like copy-pasting code I don't understand. I dig into how things work — even when it's slow — because the foundation matters more than the speed.

🚀
Ship It

Perfect is the enemy of done. I push commits, put things online, and improve in public. A working version today beats a perfect version never.
